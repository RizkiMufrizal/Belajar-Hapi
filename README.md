# Belajar-Hapi

## Run App

1. npm install
2. node app.js
3. access http://localhost:3000/api/v1/hello and http://localhost:3000/api/v2/hello

## Library

1. core framework => hapi
2. dotenv => config parameter
3. joi => validation request
4. sequelize => ORM
5. postgresql => database
